# Vector4.FromFloatArrayToRef method

Updates the given vector "result" from the starting index of the given FloatArray.

**Signature:**


|  Parameter | Type | Description |
|  --- | --- | --- |
|  `array` | `FloatArray` | the array to pull values from |
