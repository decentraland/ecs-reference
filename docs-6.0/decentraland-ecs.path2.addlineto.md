# Path2.addLineTo method

Adds a new segment until the given coordinates (x, y) to the current Path2.

**Signature:**
```javascript
addLineTo(x: number, y: number): Path2;
```
**Returns:** `Path2`

the updated Path2.

|  Parameter | Type | Description |
